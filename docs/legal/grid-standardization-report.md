## Persistence keys

Column visibility, page size and sort persist in `localStorage` under keys generated by the shared `BNDataGrid` using the grid id. Legal grids use:

- `lg.cases`
- `lg.hearings`
- (reserved) `lg.tasks`, `lg.notices`, `lg.documents`, `lg.orders`, `lg.settlements`, `lg.fees`, `lg.payment-links`, `lg.referrals`, `lg.dash.*`

## Migration recipe (apply to remaining screens)

```tsx
import { LgDataGrid, LgStatusBadge, buildLgRowActions, type LgColumnDef } from "@/components/legal/grid";

const columns: LgColumnDef<Row>[] = [
  { accessorKey: "code", header: "Code", meta: { label: "Code", pinLeft: true, width: 140 } },
  { accessorKey: "status", header: "Status",
    cell: ({ getValue }) => <LgStatusBadge status={getValue<string>()} /> },
  // …
];

<LgDataGrid<Row>
  id="lg.<screen>"
  columns={columns}
  data={rows}
  isLoading={isLoading}
  searchPlaceholder="Search …"
  toolbarFilters={[/* status, stage, officer, date range */]}
  summary={[/* KPI chips */]}
  rowActions={buildLgRowActions<Row>({ onView, onEdit, canEdit, onHistory, onDocuments })}
  bulkActions={[/* assign, export, etc. */]}
  onRefresh={refetch}
  exportFilename="legal-<screen>"
/>
```
